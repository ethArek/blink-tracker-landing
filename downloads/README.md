# Downloads folder

Put release artifacts in this folder (installers, checksums, release notes).

The landing page reads `downloads/manifest.json` to populate:

- Version string
- Download URLs
- SHA-256 checksums (copy button)

## Suggested filenames

- `DryEyeBlink-Setup-Windows.exe`
- `DryEyeBlink-macOS.dmg`
- `DryEyeBlink-Linux.AppImage`
